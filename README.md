## Log Parser

a QNAP diagnostic log parser, tested on macOS 13.2 / GNU bash version 3.2.57 

##### Usage

sh log_parser.sh [folder_name_without space]
for example:
sh log_parser.sh Q211I009382

#####  When you can use 

Quickly navigate the following information

1. Error 

2. Firmware version

3. APP information
